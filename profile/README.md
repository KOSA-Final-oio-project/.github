## 💁‍♀️ 서비스 소개
>23.11.24 ~ 23.12.21

###### OIO는 개인간 대여 중개 플랫폼입니다.
###### 당근마켓과 같은 온라인 중고거래 플랫폼과 우리에게 익숙한 정수기나 자동차 같은 대여업체를 모티브로 한 온라인 대여거래 플랫폼을 구상하였습니다.
###### 하나의 업체가 상품을 소유, 관리하는 것이 아닌 사용자 간의 거래를 가능하게 하여 더 다양한 물품을 서로 공유하고, 공유를 보다 일상에 가깝게 할 수 있도록 하고자 했습니다.
